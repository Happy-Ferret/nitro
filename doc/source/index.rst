Nitro -- Virtual Machine Introspection Framework
================================================

Nitro is a flexible framework for virtual machine introspection. It provides a
Python API for analyzing and altering the state of KVM-based virtual machines.
With Nitro, you can monitor what the machines are doing without executing a
single line of code inside a virtual machine as all the work hapens on the host.
